#parking_lot_design

### Compiling/Building and running
This command will help is creating the target jar which we can execute using the shell script
Go to the project root folder and then run: ./bin/setup.sh


### Runing the project
NOTE: Before running, please make sure you do the above setup step. Otherwise it will not run.
The project can be run as follows:

1) It accepts a filename as a parameter at the command prompt and read the commands from that file.   
   ./bin/parking_lot.sh  <input_filepath>  
   Example: ./bin/parking_lot.sh  ./file_input.txt

### Output of the project
1) It Outputs the data to terminal.
